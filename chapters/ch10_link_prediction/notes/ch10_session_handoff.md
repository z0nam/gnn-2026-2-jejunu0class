## 3. 오늘 논의된 내용 (understanding.md에 반영 완료)

- **§1.4 그래프 = 표본**: "관측되지 않은 엣지"의 암묵적 전제. 참 엣지 집합 $E^*$의 부분 관측. 결측 설정 vs 미래 설정. 노드는 완전 관측(transductive) 전제. 벤치마크는 MCAR 시뮬레이션인데 현실은 MNAR → 과대평가 가능. 음성 라벨 부재 = PU-learning. VGAE는 이 관점의 생성 모델식 형식화.
- **§1.5 휴리스틱 표기** (사용자가 직접 추가): $\Gamma$ = 이웃 집합(감마 함수 아님), $w$ = 공통 이웃 순회 변수, Adamic–Adar의 "희귀한 공통 이웃이 강한 증거" 직관.
- **§1.6 음성 엣지는 샘플링**: 개념(완전그래프 라벨링)과 구현(양성과 같은 수만 샘플링)의 구분. val/test 음성은 고정, train 음성은 매 epoch `recon_loss`가 재샘플링. test 양성이 train 음성으로 뽑힐 가능성도 원리상 존재.
- **§1.7 AUC/AP**: AUC = 무작위 양성·음성 대결에서 양성이 이길 확률. AP = 점수순으로 읽으며 양성마다 그 시점 precision 평균(상위권 오염에 민감). 손계산 예시 포함. 1:1 평가셋에선 비슷, 실전 top-k에선 AP가 더 정렬됨. "왜 둘 다 보나" 예상 질문 답 포함.
- **§2.0 인코더-디코더 계보** (별도 추가됨): GAE는 seq2seq 갈래가 아니라 오토인코더(AE→VAE) 혈통. VAE↔VGAE 일대일 대응표. 이식에서 바뀐 두 지점 — 복원 대상이 $A$만(비대칭), 디코더가 학습 없는 내적.
